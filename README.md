#### 项目介绍
游戏名字没想好……

个人本科毕业时候需要写一个论文，正好论文范围内有一个“Xx功能的App应用设计与开发”，感觉这个命题范围比较大，而且应该是包含我这个做游戏的想法，所以以此为契机，做一款自己喜欢的游戏。即使只有雏形，只要开始了这个项目，觉得可以在后期一直完善它，作为自己一个在程序之路上成长的见证。
